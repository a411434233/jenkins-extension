    2.快速新建vue后台项目

### 快速创建vue后台
    键盘命令： Ctrl+Shift+P  
    输入xs- 
    选择创建向上前端项目
    选择vue

### 快速创建uniApp项目
    键盘命令： Ctrl+Shift+P  
    输入xs- 
    选择创建向上前端项目
    选择uni-app
